Справка

Программа управления программой обработки.

Структура программы:

  • Чистка папок
    • Удаление файлов;
    • Переименование файлов;
    • Удаление пустых папок.
  • Перемещение файлов;
    • Сборка txt-файлов;
    • Рассылка исполняемый файлов.
  • Перевод txt to xlsx;
    • Каждый файл в отдельный лист;
    • Все файлы в один лист.
  • Конфиг файл;
  • Разделение файлов.
    • Разделение для оптимизации;
    • Разделение по профилям.

Вкладка Чиста папок содержит одно поле редактирования.
В нем указывается директория, в которой будут проходит сценарий.
Сценарии следующие:

  1. Удаление файлов. Удаление файлов определённого расширения в папке и подпапках.
    Расширение указывается в поле рядом с чек-боксом.
  2. Переименование файлов. Переименование файлов определённого имени в папке и подпапках.
    Имя указывается в поле рядом с чек-боксом. Новое имя файла создается по двум шаблонам:
    • Имя папки_текущее имя
    • Имя папки_дата и время из dat-файла_текущее имя
  3. Удаление пустых директорий. Этот сценарий так же проходит в указанной папке и подпапках.
Все выпоненные операции будут отображать в большом поле над кнопкой "Выполнить".

Вкладка Перемещение файлов содержит две радио кнопки, которые меняют сценарии.
При отмеченной кнопке Сборка txt-файлов выполняется поиск текстовых файлов (именно с расширением txt)
в директории, которая указана в первом поле. И все найденные файлы будут перемещены в директорию,
указанную во втором поле. Во избежание некорректной работы программы, а именно копирование с заменой,
все найденные файлы будут переименнованы по первому шаблону. Сценарий Рассылка исполняемых файлов
- копирует во все подпапки указанной директории исполняемый файл(exe), который нужно выбрать во втором поле.
В большом поле будет указана вся информация по выполненому сценарию.

Вкладка Перевод txt to xlsx выполняет операцию переноса информации из текстовых
документов в эксель-документ по двум сценариям:

  1. Каждый файл в отдельный лист.
  2. Все файлы в один лист.
В первом поле редактирования указывается путь, где будет происходить
поиск текстовых документов. Все файлы, которые будут найдены отображатся
на большом поле. Файлы, которые нужно перевести в табличный вид можно выбрать.
Вспомогательные функции отметки производят кнопки Отметить всё и Снять выделения.
Чтобы операция была выполнена, также необходимо выбрать сценарий и ввести имя
для переведённого файла. В поле под чек-босками необходимо указать место куда
будут сохранены эксель-документы.
